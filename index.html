🚀 Event Details: Git Set Code

💡 What We Do

Git Set Code is a beginner-friendly yet impactful event where we demystify Git & GitHub for developers. Think of it as your gateway to open source contributions and collaborative coding. We’ll start from the basics,what Git is, why it’s used, and how it powers almost every software project today and move step by step into practical GitHub workflows. By the end of the session, you’ll not only know the theory but also make your first real contribution.

⚡ How We Do It

  • Introduction to Git & GitHub → Learn the concepts in simple terms.
  • Installation & Setup → Get Git configured on your system with guided support.
  • First Repository → Create and initialize your own repository on GitHub.
  • Core Commands → Master add, commit, push, pull, branch, and merge.
  • Collaboration Flow → Understand forks, pull requests, and resolving merge conflicts.
  • Live Contribution Challenge → Participants will make a PR during the event, guided by mentors.
  • Q&A + Networking → Discuss, share experiences, and clear doubts directly with industry mentors.

📍 Venue & ⏰ Time

  • Venue: NUV AUDITORIUM
  • Date: 11TH OCTOBER
  • Time: 10-4
  • Duration: 6 hours of pure hands-on learning
